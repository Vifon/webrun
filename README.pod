=head1 NAME

webrun - a transparent compatibility layer with http/https protocol

=head1 SYNOPSIS

    webrun gimp http://example.com/image.png

=head1 DESCRIPTION

Prefix any command with "webrun" to automagically resolve any URLs in the argument list into the temporary files.
